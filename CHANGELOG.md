# Changelog

## 0.3.0 - 2026-03-06
- Added full multi-mode gameplay layer (`chart`, `practice`, `endless`).
- Added strict mode, mirror lanes, seeded deterministic chart regeneration, chart preview summary.
- Added calibration tap flow and metronome test mode controls.
- Added practice speed and loop range controls.
- Added mobile touch lane zones and swipe toggle.
- Added profile progression with persisted missions/stats.
- Added quality adaptation and visual intensity controls.
- Added hold/slide completion tracking in results.
- Added CI workflow and deterministic lightweight test suite.

## 0.2.0 - 2026-03-06
- Added typed note system (`tap`, `hold`, `slide`, `double`, `mine`).
- Added fever visuals, side rails, and player trail effects.
- Added advanced scoring and special-note handling.

## 0.1.0 - 2026-03-06
- Initial playable web rhythm game baseline.
